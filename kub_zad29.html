Pytania teoretyczne

1. W jaki sposób można określić właściwość danego obiektu?

2. W jaki sposób można odwołać się do pól i metod?

3. Wymień cechy charakterystyczne konstruktora.

4. Konstruowanie obiektu – przykład (3 pola i 1 metoda).

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

6. Konstruktor w JavaScript na podstawie przykładu.

7. Znaczenie słowa kluczowego new wraz z przykładem.

8. Znaczenie słowa kluczowego prototype wraz z przykładem.

Odpowiedzi

1. Określanie właściwości obiektu

Właściwość obiektu można określić na dwa sposoby:

Sposób 1 – notacja kropkowa:

nazwa_obiektu.nazwa_właściwości = "tekst";
auto.marka = "Opel Omega";

nazwa_obiektu.nazwa_właściwości = liczba;
auto.rok = 1996;
auto.cena = 25000;

Sposób 2 – notacja tablicowa:

nazwa_obiektu["nazwa_właściwości"] = wartość;
auto["cena"] = 25000;

2. Odwoływanie się do pól i metod

Do właściwości i metod obiektu odwołujemy się, podając nazwę obiektu oraz kropkę, a następnie nazwę pola lub metody.

Przykład:
console.log(samochod1.kolor);

3. Cechy charakterystyczne konstruktora

• jest wywoływany automatycznie podczas tworzenia obiektu
• ma taką samą nazwę jak klasa (funkcja)
• nie zwraca żadnej wartości
• może przyjmować argumenty
• obiekt tworzony jest za pomocą słowa kluczowego new

4. Konstruowanie obiektu – przykład

Przykład obiektu posiadającego trzy pola oraz jedną metodę:

var osoba_1 = {
nazwisko: 'Kowalski',
imie: 'Jan',
zawod: 'piekarz',
wyswietl: function() {
document.write(this.nazwisko + ' ' + this.imie);
}
}

5. Znaczenie słowa this

Słowo kluczowe this pozwala odwołać się do pól i metod danego obiektu z wnętrza tego obiektu.

Pola oraz metody w obiekcie oddzielamy przecinkami.

6. Konstruktor w JavaScript

W JavaScript zamiast klas używa się funkcji, które pełnią rolę konstruktora.

Przykład konstruktora:

function klient(nazwisko, imie, zawod) {
this.nazwisko = nazwisko;
this.imie = imie;
this.zawod = zawod;
}

Słowo this odnosi się do aktualnie tworzonego obiektu.

7. Słowo kluczowe new

Słowo kluczowe new służy do tworzenia nowego obiektu na podstawie konstruktora.

Przykład:

var osoba1 = new klient('Kowalski', 'Jan', 'kierowca');
var osoba2 = new klient('Nowak', 'Anna', 'sekretarka');

8. Właściwość prototype

Właściwość prototype służy do definiowania wspólnych metod i właściwości dla wszystkich obiektów tworzonych na podstawie konstruktora.

Przykład:

function klient() {
this.nazwisko = 'Bielski';
this.imie = 'Paweł';
}

klient.prototype.pisz_dane = function() {
document.write(this.nazwisko + ' ' + this.imie);
}

klient.prototype.zawod = 'kierowca';

Od tej pory każdy nowo utworzony obiekt na podstawie konstruktora klient posiada metodę oraz właściwość dodaną przez prototype.
